# FileShare

## Simple file sharing service

* Backend: FastApi
* Frontend: Vite + React

1. Clone repository
```shell
git clone https://github.com/Dementris/FileShare.git
cd FileShare
```
2. Build and Start Service using Docker compose
```shell
docker-compose up --build
```
